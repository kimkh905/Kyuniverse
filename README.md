# Korean English Flashcards

A simple React Native learning app built with Expo for Korean ↔ English flashcards.

## Features

- Clean Expo project structure
- Functional React components
- Local flashcard data only
- Multiple screens with navigation
- Simple progress tracking

## Screens

- Home
- Flashcards
- Quiz
- Progress

## Tech Stack

- React Native
- Expo
- React Navigation

## Project Structure

```text
src/
  components/
  context/
  data/
  navigation/
  screens/
```

## Run Locally

```bash
npm install
npm start
```

Then open the app with Expo Go or an emulator.

## Notes

- The app uses local data in `src/data/flashcards.js`.
- Progress is stored in memory only and resets when the app reloads.
